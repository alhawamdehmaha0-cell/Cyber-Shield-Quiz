# 🔐 Cyber Shield Quiz

Test your cybersecurity knowledge!

## 📋 About
An interactive Arabic quiz app designed to teach 
cybersecurity basics in a fun and engaging way.

## ✨ Features
- 50 questions across 3 difficulty levels
- Beginner / Intermediate / Advanced
- Random question selection every session
- Explanation provided after each answer

## 🛠️ Tech Stack
- TypeScript
- CSS
- HTML
